# Shortener

L'objectif était de réaliser un raccourcisseur d'url sans utiliser de package.


## Techno(s) utilisés 

* PHP (Laravel)


## Aperçu

![page accueil](/public/preview.PNG)


## Setup

```bash
# create .env (example: .env.example)

# create database

# migration
$ php artisan migrate

# install dependencies
$ composer install

# server at localhost:8000
$ php artisan serve
```
